# ph-maya

Maya (PayMaya) number validation and formatting.

## Installation

```bash
npm install ph-maya
```

## Features

- 📱 Maya number formatting
- ✅ Maya number validation

## Usage

```typescript
import { formatMaya, validateMaya } from 'ph-maya';

formatMaya('09171234567'); // '+639171234567'
validateMaya('09171234567'); // true
```
